## Аналіз результатів

### 1. Зменшення помилки при збільшенні точок

Метод Монте-Карло демонструє **стабільне** зменшення помилки при збільшенні кількості випадкових точок N. При N = 50000 та 100 експериментах відносна похибка становить лише 0.01%.

### 2. Вплив кількості експериментів

Збільшення кількості експериментів значно стабілізує результати:
- При 1 експерименті результати мають значні флуктуації
- При 10+ експериментах результати стають стабільнішими
- При 100 експериментах досягається найкраща точність

### 3. Порівняння з SciPy quad

Функція `quad` з бібліотеки SciPy дає результат **2.666666666666667**, який практично збігається з аналітичним обчисленням інтегралу (8/3). Метод Монте-Карло з усередненням при N ≥ 1000 наближається до цього значення з похибкою менше 0.5%.

### 4. Теоретична оцінка похибки

Теоретично похибка методу Монте-Карло зменшується пропорційно до 1/√N. Усереднення по M експериментах додатково зменшує похибку в √M разів. Це означає:
- Збільшення N у 100 разів зменшує похибку приблизно у 10 разів

### 5. Переваги та недоліки

**Переваги методу Монте-Карло:**
- Простота реалізації
- Універсальність — працює для складних функцій
- Усереднення стабілізує результати

**Недоліки:**
- Повільна збіжність (O(1/√N))

## Висновки

**Точність**: При N = 50000 та усередненні по 100 експериментах відносна похибка становить лише 0.01%, що демонструє високу точність методу.

**Стабільність**: Усереднення по більшій кількості експериментів значно покращує стабільність результатів.

**Порівняння методів**: Для простих інтегралів функція `quad` є значно ефективнішою (швидша та точніша). Метод Монте-Карло доцільно використовувати для багатовимірних інтегралів.
